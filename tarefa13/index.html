1 - Qual dos seguintes conceitos está diretamente relacionado à responsividade em design web?

  1. A. SEO (Search Engine Optimization)
  2. B. JavaScript
  3. C. Flexbox
  4. D. CDN (Content Delivery Network)
  5. E. Cookies

Resposta Correta:

A resposta correta é C. Flexbox.

Justificativa: Flexbox é uma técnica de layout em CSS que permite criar designs flexíveis e responsivos de maneira mais simples. Ele é fundamental para a criação de layouts que se ajustam de acordo com o tamanho e a orientação da tela, uma característica essencial do design responsivo.

2 - Qual das seguintes opções descreve corretamente o propósito das Media Queries em CSS?

  1. A. São usadas para carregar diferentes arquivos CSS com base no navegador.
  2. B. Permitem aplicar estilos CSS apenas a dispositivos móveis.
  3. C. São usadas para aplicar estilos CSS de acordo com as características do dispositivo, como largura da tela ou orientação.
  4. D. São usadas para otimizar o desempenho do site, removendo CSS não utilizado.
  5. E. Servem para aplicar estilos CSS apenas em navegadores modernos.

Resposta Correta:

A resposta correta é C. Media Queries são usadas para aplicar estilos CSS de acordo com as características do dispositivo, como largura da tela ou orientação..

Justificativa: Media Queries são uma poderosa ferramenta no CSS que permite que os desenvolvedores apliquem diferentes estilos com base em aspectos como largura e altura da tela, orientação (portrait ou landscape), e resolução. Isso é fundamental para criar layouts responsivos que se adaptam a diferentes dispositivos e tamanhos de tela.

3. Qual a melhor abordagem para garantir a acessibilidade de um site em dispositivos móveis?

  • A. Usar apenas layout fixo.
  • B. Focar no design mobile com layout fluido.
  • C. Criar apenas um layout grande para desktops.
  • D. Não testar o site em dispositivos móveis.

Resposta correta: B. Focar no design mobile com layout fluido.

Justificativa: Focar em um layout fluido garante que o design se adapte a diferentes tamanhos de tela, proporcionando uma experiência consistente e acessível.

4. O que é um layout flexível?

  • A. Um layout que se adapta a diferentes tamanhos de tela.
  • B. Um layout fixo que não muda.
  • C. Um layout usado apenas em apps.
  • D. Um layout que utiliza tabelas.

Resposta correta: A. Um layout que se adapta a diferentes tamanhos de tela.

Justificativa: Um layout flexível é aquele que responde de maneira dinâmica ao tamanho da tela, ajustando os elementos para melhor visualização em diferentes dispositivos.

5. O que significa Mobile-First?

  • A. Focar primeiro no design para desktops.
  • B. Focar no design para dispositivos móveis antes de adaptar para desktops.
  • C. Usar apenas código mobile no projeto.
  • D. Ignorar os dispositivos móveis e otimizar apenas para desktops.

Resposta correta: B. Focar no design para dispositivos móveis antes de adaptar para desktops.

Justificativa: A abordagem Mobile-First foca em criar o design para dispositivos móveis primeiro, garantindo que a experiência seja otimizada em telas pequenas, e depois adaptada para telas maiores como desktops.
