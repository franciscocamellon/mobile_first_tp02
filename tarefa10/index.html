Questão sobre Media Queries

Qual das seguintes opções descreve corretamente o propósito das Media Queries em CSS?

  1. A. São usadas para carregar diferentes arquivos CSS com base no navegador.
  2. B. Permitem aplicar estilos CSS apenas a dispositivos móveis.
  3. C. São usadas para aplicar estilos CSS de acordo com as características do dispositivo, como largura da tela ou orientação.
  4. D. São usadas para otimizar o desempenho do site, removendo CSS não utilizado.
  5. E. Servem para aplicar estilos CSS apenas em navegadores modernos.

Resposta Correta:

A resposta correta é C. Media Queries são usadas para aplicar estilos CSS de acordo com as características do dispositivo, como largura da tela ou orientação..

Justificativa: Media Queries são uma poderosa ferramenta no CSS que permite que os desenvolvedores apliquem diferentes estilos com base em aspectos como largura e altura da tela, orientação (portrait ou landscape), e resolução. Isso é fundamental para criar layouts responsivos que se adaptam a diferentes dispositivos e tamanhos de tela.
